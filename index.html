Edgeone-Pic-Api

EdgeOne Pages 随机图片 API

    这是一个基于 腾讯云EdgeOne Pages 构建的 API 服务，提供高质量的横屏和竖屏随机图片。

Star on GitHub
# GitHub 仓库
# 功能特性
「 精选图源 」: 提供高质量的ACG随机图片，图源SomeACG。
「 自动适应 」: 自动识别桌面/移动设备，返回对应比例。
「 极速响应 」: 基于边缘计算，全球网络加速。
「 开箱即用 」: FORK项目并部署即可使用，简单快速。
# API_接口
获取横屏图片 (适合桌面端):
GET /api?type=pc 或 /pc
获取竖屏图片 (适合移动端):
GET /api?type=pe 或 /pe
设备自适应:
GET /api?type=ua 或 /ua
# 使用示例
在 HTML 中直接使用： 
<!-- 根据设备自动选择图片 -->
<img src="https://pic.tr11.cn/ua" alt="随机图片">

<!-- 强制使用横屏图片 -->
<img src="https://pic.tr11.cn/pc" alt="横屏图片">

<!-- 强制使用竖屏图片 -->
<img src="https://pic.tr11.cn/pe" alt="竖屏图片">
